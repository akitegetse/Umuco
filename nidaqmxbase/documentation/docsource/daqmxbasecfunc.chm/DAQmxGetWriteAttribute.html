DAQmxBaseGetWriteAttribute

int32 DAQmxBaseGetWriteAttribute (TaskHandle taskHandle, int32 attribute, void *value);

Purpose

Reads the value of the requested write attribute.

Parameters

Input
Name Type Description
taskHandle TaskHandle The task used in this function.
attribute int32 The attribute value that this function should retrive from the provided taskHandle.
Value Description
DAQmx_Write_RegenMode Specifies whether to allow NI-DAQmx Base to generate the same data multiple times. A nonZero value indicates that the task will use regeneration. A value of zero indicates the task will not use regeneration.
Output
Name Type Description
Value void * The buffer to hold the value of the attribute. The buffer must be at least the size of an int32.

Return Value

Name Type Description
status int32 The error code returned by the function in the event of an error or warning. A value of 0 indicates success. A positive value indicates a warning. A negative value indicates an error.
