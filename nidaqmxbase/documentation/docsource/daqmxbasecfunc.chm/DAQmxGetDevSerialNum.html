DAQmxBaseGetDevSerialNum

int32 DAQmxBaseGetDevSerialNum (const char device[ ], uInt32 *data);

Purpose

Reads the value of the device's serial number.

Parameters

Input
Name Type Description
device const char [ ] The name of the device to use in this function such as Dev1.
Output
Name Type Description
Data uInt32 * The value of the serial number for the device. To compare this value with the serial number printed on your device, view the value in hexadecimal format.

Return Value

Name Type Description
status int32 The error code returned by the function in the event of an error or warning. A value of 0 indicates success. A positive value indicates a warning. A negative value indicates an error.
