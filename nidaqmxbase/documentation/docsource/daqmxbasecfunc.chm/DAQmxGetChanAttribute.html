DAQmxBaseGetChanAttribute

int32 DAQmxBaseGetChanAttribute (TaskHandle taskHandle, const char channel[], int32 attribute, void *value);

Purpose

Reads the value of the requested channel attribute.

Parameters

Input
Name Type Description
taskHandle TaskHandle The task used in this function.
channel const char [ ] The name of the channel used in this function such as Dev1/ai0.
attribute int32 The attribute value that this function should retrieve from the provided taskHandle and channel.
Value Description
DAQmx_AI_Coupling Specifies the coupling for the channel. For valid attribute values, refer to DAQmx_AI_Coupling values.
DAQmx_AI_Excit_Src Specifies the excitation source for the channel. For valid attribute values, refer to DAQmx_AI_Excit_Src values.
Output
Name Type Description
Value void * The buffer to hold the value of the attribute. The buffer must be at least the size of an int32.

Return Value

Name Type Description
status int32 The error code returned by the function in the event of an error or warning. A value of 0 indicates success. A positive value indicates a warning. A negative value indicates an error.
