DAQmxBaseGetReadAttribute

int32 DAQmxBaseGetReadAttribute (TaskHandle taskHandle, int32 attribute, void *value);

Purpose

Reads the value of the requested read attribute.

Parameters

Input
Name Type Description
taskHandle TaskHandle The task used in this function.
attribute int32 The attribute value that this function should retrieve from the provided taskHandle.
Value Description
DAQmx_Read_AvailSampPerChan Indicates the number of samples available to read per channel. This value is the same for all the channels in the task.
Output
Name Type Description
Value void * The buffer to hold the value of the attribute. The buffer must be at least the size of an uInt32.

Return Value

Name Type Description
status int32 The error code returned by the function in the event of an error or warning. A value of 0 indicates success. A positive value indicates a warning. A negative value indicates an error.
